Shadow DOM

Shadow DOM реализует инкапсуляцию DOM-дерева. В него прячется оформительская вёрстка, необходимая для создания визуально красивого контрола/виджета и т.д
http://w3c.github.io/webcomponents/spec/shadow/index.html

Поддержка ShadowDom

  • Chrome 29+
  • Opera 17+

The Extensible Web
Manifesto

The Extensible Web
Manifesto

Разрабатывать, описывать и тестировать новые высокоуровневые функции в JavaScript. Давать возможность веб-разработчикам тестировать их до внесения в стандарт. Это создает эффективное взаимодействие между разработчиками стандартов и разработчиками продуктов.
